Launch Pad 1.0

This launch pad was built to be a long standing launch pad for the program, it has a simple computer for countdown control, as well as a service tower, which for current launches with Pegasus II and Prometheus is unused because the rockets do not need this capabiliy, but it is capable of supporting this for future rockets. It has an iron flame trench for diverting the thrust, and the clamps pertain of the clamp, set to retract by default with a pin holding it up. When it it time to launch, a servo knocks the pin aside, allowing the clamp to retract.
Status: In service
In service: Nov 2021-current
